## 使用vue框架编写项目
###  执行步骤
    1.npm install
    2.npm run dev
### 制作步骤
    1.使用mui-ui的Header组件制作header区域
    2.使用mui的制作底部的tabbar区域（tabbar.html)
        + 修改tabbar的文本和图标
        + 购物车图标
          + 使用mui的icons-extra.html中的购物车图标
          + 把icons-extra.css和mui-icons-extra.ttf放到项目中合适的位置
          + 在main.js中到如icons-extra.css样式
    3.在页面中间放置router-view用来展示路由子组件
